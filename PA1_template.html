In our analysis we will use ggplot2 for plotting and dplyr for data manipulation.

require(ggplot2)
require(dplyr)

Loading and preprocessing the data

  1. We load the data using read.csv()
setwd("~/git/RepData_PeerAssessment1")
unzip('activity.zip')
data <- read.csv('activity.csv', na.string='NA', 
                colClasses = c('integer', 'character', 'integer'))
  1. We convert the date column (it should be POSIXct as dplyr doesn't accept POSIXlt, POSIXt)
data$date <- as.POSIXct(data$date)

What is mean total number of steps taken per day?

  1. Ignore NAs
complete <- data[complete.cases(data),]
  1. We summarise the data using dplyr in order to calculate the total number of steps taken per day.
summarized <- group_by(complete, date) %>% summarise(sum = sum(steps, na.rm=T))
  1. Using our summarized data frame, we can plot the histogram of the total number of steps taken each day
qplot(sum, data=summarized, binwidth=1000)

  1. We calculate the mean and median of the total number of steps taken per day
mean(summarized$sum, na.rm=T)

## [1] 10766.19

median(summarized$sum, na.rm=T)

## [1] 10765

What is the average daily activity pattern?

  1. We make a time series plot of the 5-minute interval (x-axis) and the average number of steps taken, averaged across all days (y-axis)
averaged <- group_by(data, interval) %>% summarise(avg = mean(steps, na.rm=T))
qplot(interval, avg, data=averaged, geom='line')

  1. Which 5-minute interval, on average across all the days in the dataset, contains the maximum number of steps?
averaged$interval[which.max(averaged$avg)]

## [1] 835

Imputing missing values

  1. Calculate and report the total number of missing values in the dataset (i.e. the total number of rows with NAs)
sum(1-complete.cases(data))

## [1] 2304
  1. and 3. We fill NAs by filling average value for that 5min interval
nas <- is.na(data$steps)
filled <- data
filled$steps[nas] <- averaged$avg[match(data$interval[nas], averaged$interval)]

4.

summarized2 <- group_by(filled, date) %>% summarise(sum = sum(steps, na.rm=T))
qplot(sum, data=summarized2, binwidth=1000)

In our first dataset (data) there were 8 days full of NAs, those were replaced with mean day in our second dataset. As a result we can see increase in 10k-11k category (corresponding to mean). Mean wasn't changed, while median changed to mean and variance was decreased.

Are there differences in activity patterns between weekdays and weekends?

  1. We create factor variable with 2 levels weekday/weekend.
filled$weekend <- 
    factor(weekdays(filled$date) %in% c("Sunday","Saturday"), labels=c('weekday','weekend'))
  1. We make a panel plot containing a time series plot (i.e. type = “l”) of the 5-minute interval (x-axis) and the average number of steps taken, averaged across all weekday days or weekend days (y-axis).
averaged2 <- group_by(filled, interval, weekend) %>% summarise(avg = mean(steps, na.rm=T))
qplot(interval, avg, data=averaged2, facets=weekend~., geom='line')
